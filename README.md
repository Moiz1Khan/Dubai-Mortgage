# Dubai Mortgage

Credit Link UAE mortgage website built with Next.js (App Router) and Supabase.

## Development

```bash
npm install
npm run dev
```

## Production

Set required environment variables in your host (Vercel):

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `ADMIN_EMAIL`
- `NEXT_PUBLIC_WHATSAPP_NUMBER`

Then build and deploy.
